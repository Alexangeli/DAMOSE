Class TripUpdatesService

java.lang.Object
Service.GTFS_RT.Fetcher.TripUpdates.TripUpdatesService
public class TripUpdatesService extends Object
Servizio per la gestione dei Trip Updates GTFS-Realtime. Fornisce: - fetch periodico o manuale dei trip update - gestione dello stato della connessione (online/offline) - memorizzazione dell'ultima lista di trip update in memoria La classe supporta due modalit√†: 1) Produzione: creata con URL del feed GTFS-RT, gestisce fetch automatico tramite ConnectionManager 2) Test: creata tramite dependency injection di fetcher e ConnectionManager personalizzati Autore: Simone Bonuso
