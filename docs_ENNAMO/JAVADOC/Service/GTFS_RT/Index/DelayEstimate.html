Class DelayEstimate

java.lang.Object
Service.GTFS_RT.Index.DelayEstimate
public final class DelayEstimate extends Object
Rappresenta una stima del ritardo di una corsa insieme alla sua affidabilità. Utile per combinare informazioni di ritardo provenienti da feed diversi, indicando non solo il valore stimato del ritardo ma anche quanto questa stima sia affidabile.
