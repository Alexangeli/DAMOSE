Class BestEta

java.lang.Object
Service.GTFS_RT.Index.BestEta
public class BestEta extends Object
Rappresenta la stima del tempo di arrivo (ETA) di un mezzo per una specifica corsa. Contiene sia informazioni statiche (schedule) che dati in tempo reale (realtime), insieme a eventuali ritardi e timestamp del feed. Utilizzata per confrontare e scegliere il miglior ETA disponibile per ciascun veicolo/trip.
