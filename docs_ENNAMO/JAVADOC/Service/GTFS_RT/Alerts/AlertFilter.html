Class AlertFilter

java.lang.Object
Service.GTFS_RT.Alerts.AlertFilter
public final class AlertFilter extends Object
Utility per filtrare gli alert GTFS-RT in base a una query strutturata. Questa classe consente di selezionare solo gli alert rilevanti rispetto a un determinato contesto (es. fermata, linea, trip, direzione). Il filtro è basato sugli InformedEntity presenti nell'alert: un alert viene considerato valido se almeno una delle sue entità soddisfa i criteri della query. Gli alert GLOBAL (senza entità specificate) vengono considerati sempre validi in fase di filtro; l'eventuale ordinamento per specificità viene gestito separatamente. Classe statica: non mantiene stato ed espone solo metodi utility.

    • filter

      public static List<AlertInfo> filter(List<AlertInfo> alerts, AlertQuery q)
      Filtra una lista di alert in base alla query fornita. Comportamento: - Se la lista è nulla o vuota, restituisce una lista vuota. - Se la query è nulla o vuota, restituisce una copia immutabile della lista originale. - Altrimenti, mantiene solo gli alert che soddisfano la query.
      Parameters:
      alerts - lista completa di alert
      q - query contenente i criteri di filtro
      Returns:
      lista filtrata di alert coerenti con la query

    • matches

      public static boolean matches(AlertInfo a, AlertQuery q)
      Verifica se un singolo alert soddisfa la query. Un alert è considerato valido se: - è GLOBAL (nessuna entità informata), oppure - almeno una delle sue InformedEntity soddisfa la query.
      Parameters:
      a - alert da verificare
      q - query di filtro
      Returns:
      true se l'alert è coerente con la query
