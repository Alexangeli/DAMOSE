Class DB

java.lang.Object
db.util.DB
public final class DB extends Object
Classe di utilità per la gestione delle connessioni JDBC verso SQLite.

Centralizza la creazione delle connessioni, l'abilitazione dei vincoli di integrità referenziale e fornisce metodi helper per la chiusura sicura delle risorse JDBC.

La classe è dichiarata final e non istanziabile perché contiene esclusivamente metodi statici di supporto.

    • getConnection

      public static Connection getConnection() throws SQLException
      Restituisce una nuova connessione al database SQLite su file.

      Dopo l'apertura della connessione viene esplicitamente abilitato il controllo delle foreign key tramite PRAGMA, poiché in SQLite non è attivo di default.

      Returns:
      nuova connessione JDBC pronta all'uso
      Throws:
      SQLException - in caso di errore nella creazione della connessione

    • getMemoryConnection

      public static Connection getMemoryConnection() throws SQLException
      Restituisce una connessione SQLite in-memory.

      Questa connessione non utilizza il file app.db e viene usata principalmente nei test automatici per garantire isolamento e riproducibilità.

      Anche in questo caso vengono abilitate le foreign key.

      Returns:
      nuova connessione SQLite in-memory
      Throws:
      SQLException - in caso di errore nella creazione della connessione
