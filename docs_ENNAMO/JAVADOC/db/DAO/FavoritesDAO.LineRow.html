Class FavoritesDAO.LineRow

java.lang.Object
db.DAO.FavoritesDAO.LineRow
Enclosing class:
FavoritesDAO
public static class FavoritesDAO.LineRow extends Object
DTO minimale che rappresenta una riga di "linea preferita".

Non contiene logica: serve solo per trasportare i dati dal DB al livello superiore (Controller/View), mantenendo separata la responsabilità di persistenza.
