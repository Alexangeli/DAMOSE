Class ThemeManager

java.lang.Object
View.Theme.ThemeManager
public final class ThemeManager extends Object
Gestore del tema corrente dell’applicazione. Responsabilità: - Mantiene il tema corrente - Notifica i listener quando il tema cambia - Fornisce metodi utility per applicare il tema ai componenti Swing Note: - Tutti i metodi sono statici: ThemeManager è una utility singleton - Thread-safe grazie a CopyOnWriteArrayList per i listener
