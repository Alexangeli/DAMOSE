Class AccountDropdown

java.lang.Object
View.User.Account.AccountDropdown
public class AccountDropdown extends Object
Popup utente mostrato dalla dashboard quando si clicca l'icona profilo. Responsabilità: - mostra avatar, saluto con username e due azioni principali (gestione account e logout) - mostra lo stato di connessione (online/offline) tramite un indicatore grafico - gestisce apertura/chiusura e chiusura automatica al click fuori dal popup Note di progetto: - la finestra è una JDialog senza decorazioni, con forma arrotondata tramite Window.setShape(Shape) - il ridimensionamento (uiScale) viene applicato evitando di fare pack mentre il popup è visibile per ridurre flicker
