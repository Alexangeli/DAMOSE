Class LineSearchController

java.lang.Object
Controller.SearchMode.LineSearchController
public class LineSearchController extends Object
Controller della ricerca linee (route) dalla SearchBar. Responsabilità: - Caricare e indicizzare i dati necessari alla ricerca (routes + trips). - Aggiornare la lista di suggerimenti quando l’utente digita. - Gestire la selezione di una linea/direzione e delegare l’effetto alla mappa: evidenziazione della shape (fit della linea) + attivazione layer veicoli realtime. Note di design: - L’indice (LineSearchIndex) è condiviso tra istanze tramite campi statici: routes/trips non cambiano durante l’esecuzione, quindi ha senso caricarli una sola volta. - Il parsing di trips.csv è “minimo”: estraiamo solo routeId, tripId, directionId e headsign, che sono i campi utili per costruire RouteDirectionOption. - Il controller non disegna nulla: aggiorna la UI (suggestions) e comanda la MapController.
